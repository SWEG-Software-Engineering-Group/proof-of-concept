Creare cartella Backend
- git clone https://github.com/SWEG-Software-Engineering-Group/proof-of-concept.git
- cd proof-of-concept
- git checkout Mircea
- git status (per controllare di essere veramente lì)
- cd API/SWEG-Traduzioni-api
- npm i -g serverless 
- serverless dynamodb install
- npm i (per installare tutto)
- serverless offline start

Se non funziona il comando precedente:
{
- Scaricare AWS CLI: https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html

(Linux)
- sls dynamodb install
- sudo apt install awscli

- aws configure

Creare cartella Frontend
- git clone https://github.com/SWEG-Software-Engineering-Group/proof-of-concept.git-concept.git
- cd proof-of-concept
- git checkout PoC-Cardin
- git status (per controllare di essere veramente lì)
- cd Frontend/sweg-traduzioni-backoffice
- npm i
- npm start

La parte Frontend si apre da sola sulla porta 3001 (perché sulla 3000 sta già aperto Serverless) 
Per avviare, dare Yes (y)
http://localhost:3001/index

Totale: 
Porta 3000 - Serverless Offline (va runnato per primo)
Porta 3001 - Frontend (va runnato per secondo)
Porta 8000 - DynamoDB

Per runnare tutto:
(Backend)
- Essere dentro API/SWEG-Traduzioni-api
- serverless offline start
(Frontend)
- Essere Frontend/sweg-traduzioni-backoffice
- npm start
